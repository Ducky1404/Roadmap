Roadmap Học Node.js

  • 1. Nền tảng cơ bản: HTML, CSS, JavaScript (2-3 tuần)
  • 2. Node.js và Express.js: Xây dựng backend (3-4 tuần)
  • 3. MongoDB và Mongoose: Làm việc với CSDL (2-3 tuần)
  • 4. Xác thực và bảo mật: JWT, bcrypt (2 tuần)
  • 5. Hoàn thiện dự án: Xây dựng 4 dự án (4-6 tuần)
  • 6. Triển khai: Đẩy lên GitHub và Heroku (1-2 tuần)
